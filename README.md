# Car_stock_prediction
Self Project | Tool: Sklearn, Python

Objective: Helped vehicle organization to comprehend factors that are huge in anticipating cost of vehicle,

• Directed pre-processing & analysis of exploratory data to completely comprehend the given car data index.

• Separated dataset into train and test to impede overfitting and precisely access the model.

• Trained Linear Regression & evaluated by obtaining R Squared Error value on both train and test data.

• Trained Lasso Regression & evaluated by obtaining R Squared Error value on both train and test data.

• Compared actual price vs predicted price on train & test separately for both linear and lasso regression after evaluate.
